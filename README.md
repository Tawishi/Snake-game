# Snake-game :snake:
Snake game in python
## Run the Game
1. Clone the repo
```
git clone https://github.com/Tawishi/Snake-game.git
```
2. Run the code, use whichever command works for you
```
python snakeGame.py 
```
```
python3 .\snakeGame.py 
```
3. Beat your the Highscore ! <br><br>
